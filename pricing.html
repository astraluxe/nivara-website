    Common questions

    Platform AI

    Which AI model do I get on each plan?

    Every plan — free and paid — runs on the same latest, most capable AI model, included with no API key needed. Plans do not differ by model quality; they differ only by how many tokens you get per month. Higher plans simply give you a larger monthly token budget.

    What is a token, and how does it relate to tasks?

    A token is the unit AI models use to read and write text — roughly ¾ of a word (so ~1,000 tokens ≈ 750 words). One typical “task” (a caption, a short research answer, a code review) uses around 1,000 tokens. That’s why, for example, Solo’s 4M tokens works out to roughly 4,000 tasks a month. Non-technical users can think in tasks; technical users can budget by tokens.

    Can I use my own API key on a paid plan?

    Yes — BYOK works on all plans. Connect your own Gemini, OpenAI or Claude key and use it alongside or instead of the included AI. BYOK usage never counts toward your task quota.

    What counts as an AI task?

    Roughly one meaningful AI request — writing a caption, running research, doing a code review. Simple short requests use fewer resources; deep analysis of large documents uses more. Think of it as a guide, not a hard counter.

    Automation

    What counts as an automation run?

    Each time a saved automation fires — triggered by a schedule, an email, or a file drop. Creating, editing or pausing automations uses no runs.

    Do local automations count toward my quota?

    No. Automations that run entirely on your machine (local triggers, local actions, no cloud functions) don’t consume cloud run credits.

    Mesh & devices

    What is Mesh?

    Mesh connects your devices (PC, laptop, phone) so they can share AI tasks and files over an encrypted tunnel — no cloud storage needed. Each plan includes a device limit for your Mesh network.

    Payments & billing

    Can I pay in Indian Rupees?

    Yes. Every paid plan shows both ₹ and $ prices with separate payment buttons. Indian payments go through Razorpay in INR; international payments are handled via email within 2 hours.

    Is there a free plan?
