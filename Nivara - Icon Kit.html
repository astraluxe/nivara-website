How to build the .ico / for Claude Code

      06 · pipeline
    


    

      
From SVG → Nivara.ico in three commands.

      
Windows .exe icons are multi-resolution `.ico` files containing 16, 24, 32, 48, 64, 128, 256 px frames packed together. Easiest path:


      
    
        
  1. Click Bundle all sizes above. You get nivara-icons.zip with all the PNG sizes.
    2. 
        
  2. Unzip it. Open a terminal in that folder.
    3. 
        
  3. Run ImageMagick (free, cross-platform — brew install imagemagick on macOS / winget install ImageMagick on Windows):
    4. 
      


      
$magick nivara-16-paper.png nivara-24-paper.png nivara-32-paper.png ^
        nivara-48-paper.png nivara-64-paper.png nivara-128-paper.png nivara-256-paper.png Nivara.ico
        
# produces a single multi-res Nivara.ico ready for electron-builder / Tauri / Inno Setup


      
For an Electron build, point `build.win.icon` in `package.json` at the produced `Nivara.ico`. For Tauri, drop it into `src-tauri/icons/icon.ico` and call tauri icon to generate every other platform's variant in one shot. For a quick Inno Setup installer, set SetupIconFile=Nivara.ico in your .iss script.
