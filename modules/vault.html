04Zero-log policy

We see nothing.
By design.

No DNS query logs. No browsing history. No IP addresses retained. RAM-only server architecture means nothing is ever written to disk — even if we wanted to keep records, there's nothing to hand over.

  • RAM-only — nothing is ever written to disk
  • No DNS query logs, period
  • No IP addresses stored at any point
  • Open-source client — inspect the code yourself

The four things
we cannot see.

  • No browsing history.Domains you look up via DNS are never logged anywhere.
  • No DNS queries.We process your request and immediately discard it from memory.
  • No connection timestamps.We don't store when you connected or disconnected.
  • No IP retention.Your originating IP is dropped the moment the query resolves.
