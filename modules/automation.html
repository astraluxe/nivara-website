01The canvas

Build it once.
Watch it run.

An infinite canvas where every block is a step in your business. Drag a trigger from the sidebar, drop an agent in the middle, connect it to your tools on the right. That's a workflow.

  • 5 trigger types — webhook, cron schedule, email received, form submitted, API call
  • Action library — send email, update sheet, post to Slack, run a Krew agent, call any API
  • If / else branching with conditional logic nodes
  • Loops, retries and error handlers built in
  • Variables and shared state across the whole workflow

03Human in the loop

You stay in
control.

Some moments need a human. Pause the workflow at any node, send yourself a notification, wait for approval. Before a client email goes out. Before a payment is sent. Before anything irreversible.

  • Approve / reject in Slack, WhatsApp or web
  • Set a max wait time — workflow auto-fails if you don't respond
  • Multi-approver flows with quorum rules
  • Full audit log of who approved what, when
// step 01
Workflow runs

// step 02 · paused
Pings you to approve

WhatsApp + Slack notification with the draft + accept / edit / reject.

// step 03
Sends only if approved

On accept the email ships. On reject it's discarded with reason logged.
